  # 🔍 Network Port Scanner
  
  A web-based Network Port Scanner built with Python and Flask that scans open ports on a target host and generates detailed HTML reports.
  
  ## 🚀 Features
  
  - Scan open ports on any target IP or hostname
  - Detect service names running on open ports
  - Generate downloadable HTML scan reports
  - Clean and responsive web interface
  - Fast multi-threaded scanning
  
  ## 🛠️ Technologies Used
  
  - **Language:** Python
  - **Framework:** Flask
  - **Frontend:** HTML, CSS
  - **Concepts:** Socket Programming, Port Scanning
  
  ## ⚙️ How to Run
  
  ```bash
  pip install -r requirements.txt
  python app.py
  ```
  
  Then open your browser and go to:
  ```
  http://127.0.0.1:5000
  ```
